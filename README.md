# Solana Voting App 🗳️

Un proyecto simple de votación descentralizada construido en la blockchain de Solana utilizando Anchor.

Este programa permite crear una encuesta y registrar votos para dos opciones.

---

## 📌 Características

- Crear una encuesta con una pregunta
- Votar por opción A
- Votar por opción B
- Los votos se guardan en la blockchain

---

## 🛠 Tecnologías utilizadas

- Solana
- Anchor Framework
- Rust
- TypeScript
- Solana Playground
